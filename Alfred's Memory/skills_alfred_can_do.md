**The Goal:** Build Alfred a skill set — a collection of abilities that help organize, automate, and assist in daily life.

---

## Confirmed Skills

| Skill | What it does | Status |
|-------|--------------|--------|
| **Omni-access** | Web app, available on any device | Phase 1 ⭐ |
| **Physical presence** | Robot body — sees, hears, talks, moves | Phase 2 |
| **Email automation** | Scan Outlook/Gmail, detect assignments, notify, add to to-do | Planning |
| **Notion sync** | Push tasks to Notion, organize your life | Planning |
| **Calendar guard** | Check your schedule, remind + prep you | Planning |
| **Expense tracker** | Parse bank emails, categorize spending | Planning |
| **Health nag** | Remind water, movement, sleep schedule | Planning |
| **Learning buddy** | Quiz you, track study progress | Planning |

---

## Potential Skills (Ideas)

| Skill | What it could do |
|-------|------------------|
| **Focus mode** | Block distractions, Pomodoro timer, habit tracking |
| **Document watcher** | Monitor folders (Drive, Dropbox), summarize new files |
| **Smart home control** | Integrate with Home Assistant, control lights, temp, security |

## Notes
- This is the "master resume" of what Alfred can/will be able to do
- Each skill may become its own mini-project
- Skills can integrate with each other (e.g., email automation → Notion sync → notify via web app)
